Link: OpenAI Vision Guide

2. O Fluxo de Integração (Como colocar no App)
Para o seu protótipo, o fluxo de código seria basicamente este:

Frontend (O que o usuário vê): O usuário abre a câmera no seu app, tira a foto da máquina.

Backend (Onde a mágica acontece):

Seu backend recebe essa imagem e a envia via API (para o GPT-4o ou Google Vision).

A IA analisa a imagem e devolve um JSON (um formato de texto estruturado) dizendo: "Identifiquei: Motor WEG 5CV 1990. Problema: Baixa eficiência. Sugestão: Inversor de Frequência X".

Resultado: Seu app recebe esse JSON e mostra para o usuário uma tela bonita com o "Relatório de Sustentabilidade" da máquina.

3. Exemplo Prático (Se você estiver usando GPT-4o)
Você enviaria este "Prompt" para a API junto com a foto:
